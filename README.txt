VirtStat
========

VirtStat is a shell script program that gives info from KVM hosts.


USAGE
=====

As a command: 

	$ VirStat [mail|screen|copy]

	mail) This function sends a mail with the lists of VM's in all virtualization hosts.
	
	screen) Like mail but prints in screen the information.

	copy) Copy a file or directory in the same path on all other virtualization hosts. USAGE: VirtStat copy PATH
